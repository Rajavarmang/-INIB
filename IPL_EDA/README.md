# 🏏 IPL Exploratory Data Analysis (EDA)

A data science project exploring Indian Premier League (IPL) match and player data to uncover trends, team performance patterns, and key statistical insights using Python.

---

## 📌 Project Overview

This project performs an end-to-end Exploratory Data Analysis on IPL data — covering data cleaning, univariate/bivariate analysis, and visualization of team, player, and match-level trends across seasons.

**Goals:**
- Clean and prepare raw IPL data for analysis
- Identify top-performing teams and players
- Explore trends in toss decisions, venues, and match outcomes
- Visualize key patterns using charts and graphs

## 🛠️ Tools & Libraries

- **Python 3**
- **Pandas** – data manipulation and cleaning
- **NumPy** – numerical operations
- **Matplotlib / Seaborn** – data visualization
- **Google Colab / Jupyter Notebook** – development environment

---

## 🔍 Key Steps in the Analysis

1. **Data Collection & Loading** – imported raw IPL match/player data
2. **Data Cleaning** – handled missing values, duplicates, and inconsistent formatting
3. **Univariate Analysis** – distribution of runs, wickets, matches per season
4. **Bivariate Analysis** – team vs. win %, toss decision vs. match outcome
5. **Visualization** – season-wise trends, top run-scorers, top wicket-takers, venue-wise stats
6. **Insights Summary** – key findings compiled at the end of the notebook

## 🚀 How to Run This Project

1. Clone the repository
   ```bash
   git clone https://github.com/Rajavarmang/ipl-eda.git
   cd ipl-eda
   ```

2. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```

3. Open the notebook
   ```bash
   jupyter notebook notebooks/ipl_eda.ipynb
   ```
   Or open `ipl_eda.ipynb` directly in Google Colab.
